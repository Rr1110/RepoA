# Repo Spring

## 简介

使用gradle引入相关依赖，以及支持使用gradle相关命令启动项目（将说明写在项目的Readme.md中）

引入 Spring Boot、Spring Data JPA、Flyway 等框架，实现 CRUD，包含（GET POST DELETE PUT）endpoint。

## 需求说明

### 功能需求

1. 使用Java和Spring boot实现（GET POST DELETE PUT）endpoint 各一个 
2. 涵盖参数校验，异常捕获，响应
3. TDD开发  按制定要求查询。


## 项目启动
 - 使用脚本启动
    ```./auto/dev``` 
 - 使用脚本运行测试
    ``./auto/test``
